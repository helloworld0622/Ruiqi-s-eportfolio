BlueShield: Enhancing Network Security with Intrusion Detection and Defense

Identifying, exploiting, and detecting the BlueKeep vulnerability in network traffic using Snort IDS and various security tools, ultimately enhancing network security through robust intrusion detection and defense mechanisms.

GitHub

Project Overview

In this project, we focused on enhancing network security through intrusion detection and defense mechanisms using Snort IDS and IPTables. The primary task was to identify and exploit a vulnerability in one of the target machines (MarsN2.0, MarsR2.0, or MarsY2.0), with the selected vulnerability being the “Microsoft RDP RCE (CVE-2019-0708)” or BlueKeep. This vulnerability allows remote code execution by exploiting weaknesses in the Remote Desktop Protocol (RDP) service. We used Metasploit to execute the attack and analyzed the attack sequence using Wireshark to understand the data packet transmission. Snort rules were then created to detect the attack based on specific packet signatures observed during the BlueKeep exploit.

The Snort IDS was deployed on MarsR2.0 in intrusion detection mode, and the BlueKeep attack was executed from a Kali Linux machine against MarsY2.0. The Snort rules successfully detected the attack, generating alerts for both active and inactive BlueKeep attempts.
